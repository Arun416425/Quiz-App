# Quiz App

A simple and interactive Quiz Application built using React.js.

## Features

- Multiple-choice quiz questions
- Instant answer validation
- Correct and wrong answer highlighting
- Score tracking
- Result screen
- Reset quiz functionality
- Responsive UI

## Tech Stack

- React.js
- CSS
- JavaScript

Installation
Clone the repository
git clone https://github.com/Arun416425/Quiz-App.git
Navigate to project folder
cd Quiz-App
Install dependencies
npm install
Start development server
npm run dev
How It Works
User selects an answer
App checks whether the selected answer is correct
Correct answer is highlighted
Score updates automatically
User can move to next question
Final result is displayed at the end
