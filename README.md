# Quiz App

A simple and interactive Quiz Application built using React.js.

## Features

- Multiple-choice quiz questions
- Instant answer validation
- Correct and wrong answer highlighting
- Score tracking
- Result screen
- Reset quiz functionality
- Responsive UI

## Tech Stack

- React.js
- CSS
- JavaScript
